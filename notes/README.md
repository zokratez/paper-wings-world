# Paper Wings World — Obsidian Vault

This folder (`notes/`) is the primary working memory for the entire project.

**How to use**:
- Open this `notes/` folder directly in Obsidian (File → Open Vault → choose this folder).
- It is intentionally separate from the Unity project so research, design, and long thinking stay in rich Markdown with backlinks, canvases, and queries.

## Folder Purpose

- **Research/** — Tech spikes, terrain data sources, aerodynamics references, competitor analysis, cost modeling
- **Designs/** — UI patterns, folding screen wireframes, flight HUD concepts, visual style guides
- **Game-Features/** — Detailed specs for planes, regions, modes, progression
- **Aerodynamics/** — First-principles notes, flight model requirements, wind/thermal simulation design
- **Marketing/** — Positioning, App Store copy, future website content, influencer angles
- **Roadmap/** — Vision, scope locks, phase checklists, decisions, timeline

## Current Status

All execution tracking will move to Linear (or GitHub Projects) once active development begins after Huh? Build 44.
